# ADR-007: Updates Preserve the Installation Channel

## Status

Accepted

## Date

2026-07-17

## Context

ADR-006 ships Kitten through two supported channels: a checksummed standalone binary and a global npm package. An update command must not silently replace a package-manager-owned installation with a standalone binary, or attempt to overwrite an executable whose ownership it cannot prove.

## Decision

`kitten --update` preserves the verified installation channel. A standalone installation fetches the matching latest release asset, verifies its SHA-256 checksum, and atomically replaces its binary. A global npm installation updates `@matheusbbarni/kitten` through npm. An unknown or custom installation is left unchanged and receives a precise manual recovery command.

## Consequences

- The npm launcher must provide reliable installation provenance to the compiled binary.
- The update command remains explicit and never changes channels as a side effect.
